## Pokretanje u XAMPP-u
1. Kopiraj folder `cms_project` u `C:\xampp\htdocs\`
2. Pokreni Apache + MySQL u XAMPP Control Panel-u
3. Otvori phpMyAdmin: http://localhost/phpmyadmin
4. Import SQL: `cms_project/sql/schema.sql` (kreira bazu `cms_project`)
5. Otvori projekt:
   - http://localhost/cms_project/index.php?menu=home
